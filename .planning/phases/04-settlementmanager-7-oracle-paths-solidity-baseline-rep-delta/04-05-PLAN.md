Event handler pattern (challenge-escrow.ts):
```typescript
export function handleCallSettled(event: CallSettled): void {
    let callId = event.params.callId.toString();
    let settlement = ensureSettlement(callId);
    settlement.outcome = event.params.outcome == 1 ? 'CallerWon' : 'CallerLost';
    settlement.settledAt = event.block.timestamp;
    settlement.txHash = event.transaction.hash;
    settlement.save();
    // Update Call entity
    let call = Call.load(callId);
    if (call != null) { call.status = 'Settled'; call.outcome = settlement.outcome; call.save(); }
}
```

AssemblyScript constraints (apply to all handlers):
- No closures, no float, no null for value types
- BigInt.fromI32(0) for zero-init, '' for strings, new Bytes(0) for bytes
- @entity(immutable: false) on Settlement, Dispute (status transitions)
- @entity(immutable: true) on RepEvent, ForceSettlement (append-only)
