## Decisions Made

1. **AuthKitProvider innermost.** The AST test (`privy-provider-order.ast.test.ts`) inspects the first three `JsxElement` descendants in document order and requires `PrivyProvider`, `QueryClientProvider`, `WagmiProvider`. Wrapping `AuthKitProvider` around `QueryClientProvider` would make it `jsxElements[1]` and fail the test. Placing it as the innermost wrapper (around `{children}`, inside `ToastProvider`) keeps the locked order and still satisfies CLAUDE.md "AuthKitProvider wraps below PrivyProvider." Verified green after the edit.
2. **`.test.ts` (node) over `.test.tsx` (Testing Library).** apps/web's `vitest.config.ts` has `include: ['tests/**/*.test.ts']`, `environment: 'node'`, and no `@testing-library/react` / jsdom. The plan's threat model locks "No new deps". So the AUTH-16 invariant is tested against the pure `shouldRenderFollowGraphSection()` helper (the exact "declined never renders" decision the feed section will call) plus Tier-1 static-source assertions on the page — the same pattern `onboarding.spec.ts` uses. This runs under the existing `pnpm --filter @call-it/web test follow-graph-optin` command with zero new dependencies.
3. **Follow-graph preference persistence is local + best-effort relayer.** The relayer is not in this plan's `files_modified`, and the existing `/api/onboarding/advance` route uses only a `followgraphOptinAt` timestamp under a Fastify schema that strips extra body props. The plan explicitly allowed "POST a dedicated preference." The page writes a durable per-browser copy (read by the 01.5-05 feed render gate) and best-effort POSTs to `/api/social/follow-graph-optin` (forward-compatible; the relayer route lands in the FEED wave). The AUTH-16 render gate is the load-bearing guarantee and is fully tested now.
4. **`unlinkTwitter`/`unlinkFarcaster` added to the web ABI stub.** The shared `apps/web/lib/abis/ProfileRegistry.ts` omitted the unlink functions; the settings unlink write needs them. Signatures (no-arg, `nonpayable`, user-callable) match `ProfileRegistry.sol:168-182`.
