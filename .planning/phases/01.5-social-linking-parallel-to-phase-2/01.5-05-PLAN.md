<interfaces>
X API (raw fetch, Bearer X_API_BEARER_TOKEN):
  GET https://api.twitter.com/2/users/:id/following  → { data: [{ id, username }] }  (follows.read)
Neynar (raw fetch or @neynar/nodejs-sdk@3.176.0 — gated; prefer raw fetch):
  GET https://api.neynar.com/v2/farcaster/following?fid=<fid>  (header x-api-key: NEYNAR_API_KEY)
Feed section response: { items: Array<{ callId, handle, marketLine, status, deadline }>, source: 'live'|'cache'|'empty' }
Content rules (AUTH-15): ≤10 items, recency desc, status ∈ {live, active-duel}, EXCLUDE settled.
</interfaces>
