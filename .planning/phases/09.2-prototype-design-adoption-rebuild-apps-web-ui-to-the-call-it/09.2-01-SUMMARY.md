## Decisions Made

- Donor `.uppercase` utility (text-transform + letter-spacing 0.08em) deliberately NOT ported — it would override Tailwind's `.uppercase` class globally and silently restyle existing markup; the rest of the donor utility vocabulary (`.row`/`.col`/`.spread`/`.mono`/`.muted`/`.text-*`/`.bg-cream`/`.slot-num`) is in
- Donor companions `.nav-kbd`, `.chip-row`, `.text-secondary`, `.text-tertiary` ported alongside their enumerated siblings (part of the donor vocabulary, zero collision risk)
- `* { border-radius: 0 }` lands as a zero-specificity default (D-01) — explicit `rounded-*` utilities/inline styles still win until later plans rewrite that markup
- Page-level grid classes (`.layout`, `.app-header`, `.sidebar`, `.main`, `.brand`) NOT ported — they land with the shell in plan 09.2-03 as specified
