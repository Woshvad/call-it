## Accomplishments

- **AppShell.tsx**: composes optional SystemTicker → `.app-header` (sticky, 64px, 3-col grid: brand "CALL IT · be right in public" / `.brutal-search` visual placeholder with ⌘K kbd / NotificationBell + wallet pill) → `.layout` 240px/1fr grid with `.main` (max-width 1180, `0 32px 80px`). Sticky offsets are DERIVED: AppShell sets `--shell-offset` to `32px`/`0px` from `NEXT_PUBLIC_SYSTEM_TICKER === '1'`; `grep -c "96px" AppShell.tsx` = 0 (D-11). `/signin` and `/onboarding/*` render full-bleed children with no shell. Below 768px (useIsMobile) the sidebar hides and an `.icon-btn` hamburger (44px touch target) opens the existing MobileDrawer. Static chrome renders unconditionally; only the wallet pill/bell/rep card gate on Privy (late-hydration stability, wiring-risk #4)
- **Wallet pill (AUTH-44)**: "2,840.21 USDC · handle" from the existing `useUsdcBalance` (5s poll + Transfer-event refetch untouched) + `useProfile` viewer fetch; renders the handle ONLY when the relayer's handle source is not `'truncated'` — otherwise balance alone; zero address-formatting helpers in the shell (acceptance grep = 0)
- **Sidebar.tsx**: `// TAPE` (The Tape `/`, Leaderboard) + `// YOU` (Make a call, Your profile `/profile/<viewer>`, Settings, Disputes) as Next Links with pathname-derived `.nav-item.active`; profile/settings links hidden until `authenticated && ready && address` (never flash authed links). Cream `.sidebar-rep` card: 44px Archivo 900 `globalRep` + "{acc}% accuracy · {n} settled" line from the existing relayer profile fetch — hidden entirely when no profile data (D-07). `.sidebar-meta` footer: `v0.0.1 · ARBITRUM SEPOLIA` (version imported from package.json; no fake block number, D-05)
- **SystemTicker.tsx**: `.system-ticker` 32px recipe behind the flag, default OFF; `items` prop defaults to `[]` with NO mock prices and NO fetch (D-05/T-09.2-09) — `TickerItem` contract documented for the future price feed

## Decisions Made

- **Duel nav item cut**: the plan mentioned the prototype's orange `.nav-dot` on a Duel item, but no `/duel` index route exists (only `/duel/[challengeId]`) — shipping it would be a dead control (D-08). The `.nav-dot` recipe remains in CSS for a future duel surface
- **`'truncated'` handle source = no handle** in the wallet pill: the relayer falls back to a shortened address as the "handle"; rendering it would violate AUTH-44, so the pill shows balance-only in that case
- **MobileDrawer labels** aligned to the new sidebar voice (The Tape / Make a call / Your profile) — grep confirmed zero tests referenced the old labels
- **Search input** ships as a readOnly visual placeholder with ⌘K kbd hint — explicitly sanctioned by D-10
