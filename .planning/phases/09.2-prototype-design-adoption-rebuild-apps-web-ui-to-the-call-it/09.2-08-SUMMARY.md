## Decisions Made

- **Accents never sit as raw text on cream:** chartreuse/red/warning/purple all fail contrast on #F5F1E8 as text. The pattern is black chips/strips carrying the accent as text, or accent-filled buttons with black text and black borders. This is now the cream-surface reference pattern
- **Radix translate-centering kept** in the three packages/ui modals instead of restructuring to the recipe's flex-centered overlay — the five template values are what define the look; restructuring Radix Overlay/Content would risk focus-trap/aria behavior for zero visual gain
- **Error palette on cream** darkened to #DC2626/#B91C1C (the dark-theme #F87171 is 2.2:1 on cream)
- **Quick-stake active state** derived render-only from existing `stakeInput` state — toggle-pill `.on` semantics with zero new state or handlers
