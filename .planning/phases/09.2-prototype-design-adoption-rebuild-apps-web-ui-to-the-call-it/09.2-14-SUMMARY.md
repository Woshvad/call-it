## Decisions Made

- **Satori literal mirroring:** OG markup cannot consume `var(--*)` — the fix uses the token's literal value with a comment naming the source var, so future token changes have a greppable trail
- **ProfileTabs kept (token-swapped) despite being unused by app code** — profile-shell.spec Test 7 asserts its existence; deletion would require a lockstep spec change outside this plan's audit mandate
- **Make a call always-visible in the drawer:** the Sidebar (the design canon since 09.2-03) renders it unconditionally; `/new` is middleware-guarded so a logged-out tap redirects to /signin — the Pitfall-5 "never flash authed links" rule applies to viewer-identity links (profile/settings), which stay gated
- **signin 80px decorative glyphs left unclamped** — outside the 56-132px display-role mandate's overflow failure mode (1-2 glyphs ≈ ≤100px wide at 375px)
- **OG Archivo refresh deferred to 09.1** (D-16 optional item; D-04 divergence stands)
